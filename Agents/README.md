# Leo Python Agents

This folder contains Python-based AI agents for deep-diving into company data.

## Folder Structure
- `src/`: Core agent logic and researcher tools.
- `config/`: Configuration and environment variables.
- `utils/`: Common utility functions.
- `main.py`: Entry point for running agents.

## Getting Started
1. Create a virtual environment: `python -m venv venv`
2. Activate it: `source venv/Scripts/activate` (Windows)
3. Install dependencies: `pip install -r requirements.txt`
